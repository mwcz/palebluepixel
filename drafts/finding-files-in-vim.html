Drawbacks

There's no such thing as a perfect method (for doing anything), and this one has a few drawbacks.

If you like using tabs in gvim, this probably won't work. It opens every file in the project, and vim won't open many tabs at once. The command :tabnew ** in my 500-file project results in: . I typically use vim inside Terminator, and use multiple buffers in a single window, instead of in tabs. If you do use gvim, a nice tab-like option is the Buffers menu.

It opens all files, including binary files. You'll never see any of them unless you switch to its buffer.
