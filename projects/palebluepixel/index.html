Pale Blue Pixel

Pale Blue Pixel is this blog. I write technical posts about web development, image processing, amateur graphics, and generally celebrating the art and craft of programming. The name Pale Blue Pixel is a meek tribute to Carl Sagan's Pale Blue Dot.

It's currently powered by:

  • Python - a large and powerful snake
  • Pelican - an awkwardly-shaped sea bird
  • CSS Grid - framework-free CSS thanks to grid and flexbox
  • LESS - CSS preprocessor
  • GitHub pages - hosting
